
# 🌍 Travel and Analytics Dashboard (Power BI)

---

The project focuses on visualizing important tourism indicators such as **Tourism Arrivals, Tourism Receipts, GDP Contribution, and Tourism Growth Percentage**. Through interactive charts and KPI cards, users can easily explore tourism performance across different countries and years.

---

## 📊 Overview

Travel and Analytics Dashboard is an interactive data visualization project developed using **Microsoft Power BI** to analyze global tourism trends and their economic impact.  
This dashboard integrates tourism datasets from reliable sources such as **UN Tourism, World Bank, and Kaggle** to provide insights into international travel patterns, tourism revenue, and economic contributions.

---

## 🎯 Objectives

The main objectives of this project are:

• Analyze global tourism trends using tourism datasets.  
• Visualize tourism data in a clear and interactive format.  
• Identify patterns in tourist arrivals and tourism revenue.  
• Understand tourism's economic contribution through GDP indicators.  
• Compare tourism performance across countries and time periods.

---

## 🛠️ Tools and Technologies Used

➤ **Microsoft Power BI** – Data visualization and dashboard creation  
➤ **Power Query** – Data cleaning and preprocessing  
➤ **Data Modelling** – Create relationships between the key metrics  
➤ **Dataset** – World tourism dataset
